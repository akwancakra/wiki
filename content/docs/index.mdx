# Fumadocs Dummy

Ini adalah halaman dummy dokumentasi untuk testing Table of Contents (TOC) dan layout.

---

## Pengantar

Fumadocs adalah sistem dokumentasi modern berbasis MDX & React.

### Fitur Utama

- Mendukung komponen React
- Syntax highlighting
- Table of Contents otomatis

---

## Instalasi

```bash
npm install fumadocs
```

---

## Penggunaan Dasar

### Import Komponen

```jsx
import { DocsWrapper } from "fumadocs";
```

### Contoh Render

```jsx
<DocsWrapper>
  <h2>Contoh</h2>
</DocsWrapper>
```
